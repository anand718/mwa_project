## 🚀 Project Overview
**CyberLab** is a high-fidelity, immersive threat simulation dashboard designed for educational purposes. It provides a "safe-sandbox" visualization of common cyber-attacks, allowing students and professionals to understand the mechanics of Ransomware, Viruses, and Remote Access Control (RAC) without any actual risk to their systems.

### 🧪 Core Simulations
1.  **Ransomware (AES-256 Simulation)**: Visualizes the file encryption process and the subsequent ransom demand.
2.  **Polymorphic Virus**: Demonstrates the rapid self-replication and system-wide spread of malicious code.
3.  **RAC (Remote Access Control)**: Simulates the establishment of an unauthorized SSH tunnel and remote file system access.

---

## ✨ Key Features
- **Immersive 3D-Feel UI**: Built with a futuristic "Mission Control" aesthetic using glass-morphism and neon accents.
- **Real-time Terminal**: A live-scrolling system log that provides technical feedback for every action.
- **Educational Insights**: Each simulation concludes with a "Prevention & Defense" guide to reinforce cybersecurity best practices.
- **Resource Monitor**: Simulated CPU, RAM, and Network load tracking to show system impact during attacks.

## 📦 Installation & Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/cyberlab-simulator.git
   cd cyberlab-simulator
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
